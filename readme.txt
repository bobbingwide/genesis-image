Genesis Child Theme with image.php


== Description ==
genesis-image is a child theme for the Genesis Theme Framework

* Includes the image.php template file to display images

== Installation ==

1. Purchase and Install the Genesis parent theme
2. Download the genesis-image child theme
3. Install the genesis-image child theme
4. Activate the genesis-image child theme
5. Navigate to Appearance > Customize in your admin panel and customize to taste.
6. Navigate to Genesis > Theme Settings to set other options
7. Optionally, install and activate a range of oik-based plugins.
